## 色标应用细则
#### ‌色阶过渡原理‌

暖色系（红→橙→黄）对应高遗忘风险期，需高频刺激‌15

冷色系（绿→蓝）反映记忆稳定度提升，降低复习密度‌8

#### ‌动态调整机制‌

连续三次正确回忆后，色标降阶处理（如🔴→🟠）‌57

错误率＞20%时逆向升阶（如🟢→🟡）‌6
